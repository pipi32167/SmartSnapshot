为什么浏览器测试需要打包？

remark 采用 ESM 模块化设计，依赖较多，直接使用 CDN 会比较复杂。

推荐使用方式：

  1. Node.js 环境（推荐）
  2. 使用 Vite/Webpack 打包
  3. 使用 Skypack/esm.sh 等 ESM CDN

适用场景：

  • ✅ 需要操作 AST
  • ✅ 复杂文档转换流程
  • ✅ 需要 Lint 检查
  • ❌ 简单的 Markdown 渲染（体积过大）
