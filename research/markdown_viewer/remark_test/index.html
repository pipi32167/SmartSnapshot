📝 remark / unified 测试

库信息： unified v11.x | AST 处理 | ~80KB+ gzip | 🌲 生态最强（500+ 插件）

为什么浏览器测试需要打包？

remark 采用 ESM 模块化设计，依赖较多，直接使用 CDN 会比较复杂。

架构说明

unified (框架)
  ├── remark → Markdown
  ├── rehype → HTML
  ├── retext → 自然语言
  └── redot  → Graphviz

适用场景

  • ✅ 需要操作 AST（抽象语法树）
  • ✅ 复杂文档转换流程
  • ✅ 需要 Lint 检查
  • ❌ 简单的 Markdown 渲染（体积过大）
